## PS Prep

### 목표 1 (Lv1~Lv2)

* 0x01~0x02 기초 코드 작성 요령 (입출력/구현 습관)
* 0x03 배열
* 0x04강 - 연결 리스트
* 0x05 스택 / 0x06 큐 / 0x07 덱
* 0x08 스택 활용(괄호/수식)
* 0x09 BFS / 0x0A DFS
* 0x0B 재귀 / 0x0C 백트래킹
* 0x0D 시뮬레이션(구현)
* 0x0E~0x0F 정렬
* 0x13 이분탐색
* 0x14 투 포인터
* 0x15 해시
* 0x11 그리디
* 0x10 DP (기본)

### 목표 2 (Lv1~Lv2)

* 0x17 우선순위 큐 (힙)
* 0x18 그래프
* 0x19 트리
* 0x1D 다익스트라
* 부록 D Union-Find

### 목표 3 (Lv3)

* 목표 1~2 범위 응용(변형 문제 포함)

### 목표 4 (선택 심화)

* 0x1A 위상정렬 / 0x1B MST / 0x1C 플로이드
* 0x1E KMP / 0x1F 트라이
* 부록 C 비트마스킹 / 0x12 수학
* 부록 E DP 심화
